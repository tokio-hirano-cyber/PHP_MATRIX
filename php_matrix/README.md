# エンジニア管理システム

さくらインターネットレンタルサーバー用のPHPエンジニア管理システムです。

## 機能

- ✅ ログイン・ログアウト機能
- ✅ エンジニア一覧表示
- ✅ エンジニア追加（CREATE）
- ✅ エンジニア更新（UPDATE）
- ✅ エンジニア削除（DELETE）
- ✅ 検索機能（名前、メール、スキルで検索）

## セットアップ手順

### 1. データベースの準備

1. さくらインターネットのphpMyAdminにアクセス
2. `database.sql` の内容を実行してテーブルとダミーデータを作成

### 2. ファイルのアップロード

すべてのPHPファイルをさくらインターネットのサーバーにアップロードしてください。

### 3. アクセス

ブラウザで `https://your-domain.com/login.php` にアクセス

## ファイル構成

- `config.php` - データベース接続設定
- `login.php` - ログインページ
- `logout.php` - ログアウト処理
- `index.php` - メイン画面（一覧・検索）
- `create.php` - 新規追加ページ
- `edit.php` - 編集ページ
- `delete.php` - 削除処理
- `database.sql` - データベーステーブル作成SQL
